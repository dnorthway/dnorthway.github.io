Install App

(Note: For Mac users; right click the Installation File and Data Template links below and select Download Linked File As.)

Installation: (OneDrive) Right click and save this Installation File. to your computer. View your OneDrive online. Select New; Excel workbook. On the Home ribbon, select Add-ins, select Manage My Add-ins, select Upload My Add-in then upload the 'manifest.xml' file to install the app. Click the Save Data App button, the task pane will appear. Select New Project.

Installation: (Dropbox) Right click and save this Installation File. to your computer. Upload this Data Template. to your Dropbox and open it in Excel for the web. On the Insert ribbon, select Add-ins, select Manage My Add-ins, select Upload My Add-in then upload the 'manifest.xml' file to install the app. Click the Save Data App button, the task pane will appear.

For additional installation instructions, see (Manually sideload an add-in to Office on the web).

Desktop Installation: (Sideload an Office Add-in) Excel 2013 or later required.

How to Seamlessly Integrate Data with this app

Welcome to our tutorial on harnessing the power of Data to streamline your business management using spreadsheets. Follow these steps to make your workflow more efficient:

Step 1: Adding Data

If you use other spreadsheets (Excel Worksheet, Google Sheet, even HTML) you can easily add them to your file:

  • Locate the file you want to integrate, for example, an Invoice form.
  • Copy and paste data from the file directly into the green shaded area of the Input sheet begining at cell A3. Use the 'Paste Column Width' function if available.
  • Merge all cells that contain data.
  • Copy and paste data from the Input sheet directly into the green shaded area of the View/Print sheet begining at cell A3. Use the 'Paste Column Width' function if available.

Step 2: Setup the Log table

  • On the Input sheet select the 'Log 1' cell then press '=' and select the cell of your data that contains the label of the unique ID of your data (ie.Invioce No., PO No., Report No. ect.) press 'Enter'.
  • Select the cell below Log 1 cell and press '=' then select the cell of your data that contains the data of the unique ID. Press 'Enter'.
  • Repeat this step for up to 12 table fields.

Step 3: Save, View, Print and Update your data.

  • Select the Save Data App and press Save. (Note: This will take several moments.)
  • Select the record from the green shaded dropdown box on the View/Print sheet.
  • View the Log and Update sheets.
  • To print, select the cells on the View/Print sheet you would like to print. Select 'File' then 'Print'. In 'Page setup' select 'Print' - 'Current selection'.

    • Step 4: Change the entries on the Input sheet and save to create another record.
